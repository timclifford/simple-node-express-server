# Simple-Node.js-Server

Simple Node.js Express server ready to use.

## Install

### Install all dependencies

```
npm install
```
or
```
yarn
```

## Usage

### Start project on local server (go to localhost:1337)

```
npm start
```
or
```
yarn start
```

### Build project

```
npm run build
```

### Preview build (go to localhost:1337)

```
npm run serve
```

### Run test
```
npm run test
```

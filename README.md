# Simple-Node-Express Server

Simple Node.js Express server ready to use.

## Install

### Install all dependencies

```
npm install
```
or
```
yarn
```

## Usage

### Start project on local server (go to http://127.0.0.1:3000)

```
npm start
```
or
```
yarn start
```

### Build project

```
npm run build
```

### Preview build (go to http://127.0.0.1:3000)

```
npm run serve
```

### Run test
```
npm run test
```

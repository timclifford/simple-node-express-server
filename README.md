## Install

### Clone this repo:

```
git clone https://github.com/alexdevero/react-mobx-react-router-boilerplate.git
```

### Install all dependencies

```
npm install
```
or
```
yarn
```

## Usage

### Start project on local server (go to localhost:1337)

```
npm start
```
or
```
yarn start
```

### Build project

```
npm run build
```

### Preview build (go to localhost:1337)

```
npm run serve
```

### Run test
```
npm run test
```
